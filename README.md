## Description
[]: {{{1

  venv - virtualenv's bundle exec

  venv is the virtualenv [1] equivalent of `bundle exec` [2], allowing
  you to run `venv some command` from e.g. a script, instead of having
  to `. .venv/bin/activate`, then run the command.
